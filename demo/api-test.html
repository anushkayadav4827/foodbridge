🚀 How to Start the Server

To test the API endpoints, you need to start the backend server:

1. Open terminal in the backend folder:
cd backend

2. Install dependencies (if not done):
npm install

3. Start the development server:
npm run dev

4. Server will start on:
http://localhost:3005

Once the server is running, refresh this page and the test buttons will work!
