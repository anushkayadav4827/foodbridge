## 🐛 TROUBLESHOOTING

### Page not loading?
- Check if servers are running
- Backend: http://localhost:3000/health
- Frontend: http://localhost:3001/

### Need to restart?
```bash
# In backend folder
npm run dev:mock

# In web folder
npm run dev
```

### Clear browser cache
- Press `Ctrl + Shift + R` (Windows)
- Press `Cmd + Shift + R` (Mac)
