## 🎭 Animations

### Landing Page
- **Hero Section**: Fade in from bottom (0.8s)
- **Stats Counter**: Count up when scrolling into view
- **Feature Cards**: Stagger fade in (0.1s delay each)
- **How It Works**: Scale up on scroll

### Cards
- **Hover**: Transform translateY(-4px) + shadow increase
- **Click**: Scale down slightly (0.98)

### Buttons
- **Hover**: Shadow increase + slight scale (1.02)
- **Loading**: Spinner animation

### Page Transitions
- **Route Change**: Fade in new page (0.3s)

## 📱 Responsive Breakpoints

```
Mobile:    0px - 600px   (xs)
Tablet:    600px - 900px (sm)
Desktop:   900px - 1200px (md)
Large:     1200px - 1536px (lg)
XL:        1536px+ (xl)
```

### Responsive Behavior
- **Sidebar**: Drawer on mobile, persistent on desktop
- **Grid**: 1 column (mobile) → 2 columns (tablet) → 3-4 columns (desktop)
- **Typography**: Scales down on mobile
- **Spacing**: Reduced padding on mobile

## 🌈 Visual Hierarchy

### Landing Page
```
1. Hero (Largest, most prominent)
   ↓

   ↓

   ↓

   ↓

   ↓

## 🎯 Key Visual Elements

### Badges
```
🌟 First Meal       (Green chip)
🔥 5 Day Streak     (Orange chip)

```

### Status Indicators
```
● Available         (Green dot)
● Claimed           (Yellow dot)
● Completed         (Blue dot)
● Expired           (Gray dot)
```

### Icons
- Material Icons throughout
- Consistent 24px size
- Color-coded by context
- Always paired with text labels

## 📸 Image Guidelines

### Photos
- **Aspect Ratio**: 16:9 or 4:3
- **Border Radius**: 12-16px
- **Shadow**: Soft, subtle
- **Hover**: Slight zoom (1.05)

### Avatars
- **Size**: 40px (small), 56px (medium), 80px (large)
- **Shape**: Circle
- **Fallback**: First letter of name on colored background

## 🎨 Spacing System

```
xs:  4px   (tight spacing)

xxl: 48px  (section spacing)
